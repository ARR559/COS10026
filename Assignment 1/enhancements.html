HTML Enhancements

Flip Box

This allows the user to hover over an element, before the object appears to "flip around" showing related information. This is done in combination with the CSS, overlapping both elements, before using the css indicator :hover. The indicator just edits the opacity of the elements. When one particular needs to be shown, the opacity of that element is set to 1, while the opacity of the other is set to 0. If the other element is now selected ( through the :hover ), then the opacity of the element is set to 0, while the opacity of the other is set to 1. This is useful because it allows information on the page to be condensed into a smaller space, so it does not overwhelm any potential users.

Where I use it:Lines 32-74 (jobs.html) and Lines 246-319 (style.css) on the jobs description page

Reference: https://www.freelancer.com/articles/web-development/13-cool-tricks-for-html

Light mode dark mode button

The light/dark mode button is a button which the user can press to change background colour of the page to black and the text to white. This is a popular choice by users among other applications as it reduces their eye strain by having less bright colours on the page. This enhancement uses a ⁢ button onclick = " myFunction() ">Toggle dark mode </button > to create the 'Toggle dark mode' button and change from the pages background-color and text-color to the new black background and white text. A <script> imbeds the executed code which is "myFunction()" and also uses var element = document.body to make the function retrieve and execute the darkmode class list when the "MyFunction()" is toggled. This is done by element.classList.toggle("dark-mode");.

CSS Enhancements

Fade-in transition animation

The Fade-in transition animation enhancement located on the homepage (among other pages) allows the first main image the user sees to seamlessly load in rathert than abduptly flashing in. This is done so by making the background colour animation: fadeInAnimation ease 1s; which makes the fade be 1 second of which is greater than the image load in time. Furthermore, making the background color: #FFFFFF the start of the load in camouflages in with the white page load in. Moreover, this extension allows the customisability of elements undiscussed in this unit such as the animation-iteration-count: 0, and before and after FadeInAnimation Opacity.

Where I use it: Lines 63-78 (style.css) and on every webpage

Reference: https://stackoverflow.com/questions/11660710/css-transition-fade-in

Hovering underline animation for navigation tab

When the user hovers over the underlined text on the navigation bar (which hyperlinks to another page in the website), the intitial white underline changes to a rainbow colour spectrum which is between the colours yellow, blue and light purple. This makes it easier for the user to distinguish which page link their mouse is hovering over on the navigation tab, making it easier for them to choose the right page. This is achieved by using .header_link and creating 2 underlines over each other given by linear-gradient and the using transition: background size 400ms which is duration of the shrinking of the white underline. Furthermore focus and hover are used to shrink the background-size of the linear-gradient thus shrinking the white underline and thus showing the rainbow underline.

Where i use it: Lines 10-36 (style.css).

All page elements resize to proportionally fit users changed window size

This enhancement allows all the page elements such as text, image and table size to proportionally fit within the users window size whether it is 50% of their screen or 30%. This improves user experience by having them be able to clearly view the website if they have 2 applications open and are using a split screen on their window. This has been achieved by using relative sizing such font-size: 10em or width = "50%" height = "50%" rather than using absolute metrics such as px.

Where i use it:Used on all elements

Reference: https://stackoverflow.com/questions/36054018/html-resize-text-to-keep-same-proportion-with-an-image
